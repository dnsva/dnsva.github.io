## Running Locally

No build step required. Just open `index.html` in your browser, or serve the repo with any static file server:

```bash
# With Python
python3 -m http.server 8000

# With Node.js (npx)
npx serve .
```

Then visit `http://localhost:8000`.

## Deployment

The site is automatically deployed via [GitHub Pages](https://pages.github.com/) from the `main` branch.
