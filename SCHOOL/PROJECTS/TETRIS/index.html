INSTRUCTIONS (IMPORTANT):
When you download, YOU HAVE TO CONVERT THE EXECUTABLE INTO AN ACTUAL EXECUTABLE (otherwise it will open in a text file).
STEPS:
1. Open your terminal (app)
2. You have to navigate to where you downloaded the file (usually to DOWNLOADS)!
Use the command ls to list all the files in the directory youre CURRENTLY in
Use the command cd to CHANGE DIRECTORY
Your goal is to change your directory to where you downloaded the file
3. ONCE YOURE THERE, just type this: chmod +x tetris-for-mac and you're finished. You can now open the file normally.
Note: on mac if you get a message saying that the program can't be opened, go to settings > privacy & security, scroll down & click "Open Anyway"
