Converting

--- Binary to Decimal
Add up all the values with 1s:
1001 1011 -> 1 + 2+ 8 + 16 + 128 = 15510
Binary to Hexadecimal
binary -> decimal -> hex
0110 0110 -> 2 + 4 + 32 + 64 = 10210 = 6616
TRICK: split in half
0001 0111 -> [0001 = (1)], [0111 = (7)] -> 17
0011 1010 -> [0011 = (3)], [1010 = (10)] -> 3A
Binary to ASCII
binary -> decimal -> ASCII
0010 1001 -> 1 + 8 + 32 = 41 = ')'
Decimal to Binary
Turn on all powers of 2 that are used
42 -> 0010 1010

Computer Parts

Motherboard - The main circuit board of a computer that connects all of the other components together (CPU, RAM, power supply, etc). It provides the electrical connections and allows communication between them.

Storage Device (SSD, HDD) - The part of a computer that allows data to be stored.
SSD (solid-state drive) - faster, more expensive, lower capacity, more durable
HDD (hard disk drive) - slower, cheaper, higher capacity, stores data on spinning disks.
powers of 2
2TB - lots
512GB/1TB - average
256GB - not that much

CPU (processor) (central processing unit) - Brain of the computer. Executes instructions, processes data, performs calculators. Usually 2-16 cores. Communicates with other components to complete certain tasks.
Image procesing/photoshop/etc. - 24 cores (lots)
Average - 12 cores
Normal - 8 cores

RAM (random access memory) - Short term memory. Type of memory used for temporary storage that the CPU is actively using. RAM allows the CPU to access data quickly. The more RAM you have, the better your performance usually is.
32GB - lots
16GB - average
8GB - ok

Operating System (OS) - Software that manages a computer's resources. Provides an interface for users to interact with. Allows applications to run. Windows, Linux, MacOS.

Display Device (Monitor) - Output device that displays information processed by the computer. Shows visual output (text, images, videos, grpahics). Come in different sizes, resolutions, and types.

GPU (Graphics Card) (graphics processing unit) - Performs complex calculations required for rendering graphics. Responsible for accelerating image and video processing. GPUs are used in applications such as gaming, video editing, and scientific research.

Viruses

A virus is a piece of code that attaches intself to a file. The file is now said to be infected. In addition to the ability to attach to a file, the virus generally carries a payload, which is a small program that executes when the infected file is accessed.

The payload can be harmless (e.g., a humorous message) or very malicious (e.g., erasing the entire hard drive)

Worms

  • Similar to a virus but it is more aggressive about spreading
  • They use the fact that most computers are interconnected (networks, internet)
  • They search for connections between computers and try to put copies of themselves on vulnerable systems

Trojan Horse
  • A program that looks like something you would want
  • The goal is to TRICK you into downloading and running the program
  • As soon as the program executes, the payload is delivered. This will quite often be a virus or worm, and you just willingly installed it

Spyware
  • Range of behaviour for this type of software
  • It will track your browsing habits and send targeted advertisments when you go to certain websites
  • Most web browsing involves this to some extent in the form of cookies which are legitimate files stored while browsing
  • Some types of spyware will garther information to send back to the owner/creator
  • In addition to browsing, they might include software found on your computer, personal info, private & financial info

Malware
  • Everything above is collectively referred to as malware (malicious software)

Anti-Viruses

Tools to combat viruses, worms, trojan horses, etc. = anti-virus (software)

Anti-viruses software runs on your computer at all times and actively scans files for virus signatures, and if a virus is detected, it is neutralized (cured) or quarantined (incurable)

HOWEVER,
- performance problems can be caused from having a program run all the time
- "false positives" can happen where anti-virus software will incorrectly identify a legitamate program as a virus
- a poor, old, out of date program might not recognize a new virus

Spyware Cleaners - There are software products that will find and remove spyware already installed on a system. Similar to anti-virus.

A firewall is a piece of software that acts as a barrier between a computer and the rest of the world. It can act as a filter that allows the authorized flow of data over specific channels (ports) on the computer system. They can prevent unauthorized software (e.g., worms, spyware) already on the computer from communicating with the outside world
